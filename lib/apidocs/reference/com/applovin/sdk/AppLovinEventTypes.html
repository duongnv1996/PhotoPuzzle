public static final String USER_COMPLETED_IN_APP_PURCHASE

Event signifying that the user completed an iTunes in-app purchase using StoreKit. Note that this event implies an in-app content purchase; for purchases of general products completed using Apple Pay, use kALEventTypeUserCompletedCheckOut instead.
